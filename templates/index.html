(web-lisp) is a web micro-framework written in ANSI Common LISP, currently in the very early stages of development. (web-lisp) is built on top of Hunchentoot and Clack, and directly depends on clack and cl-ppcre.

Check Getting Started for a brief overview of using this framework.

Some things that have been completed so far are:

  • Regex-based URL Routing
  • Custom templating allowing for in-line LISP expression evaluation
  • Environment Variables available on page

Here are some examples included with the package:

  • Plain Text Output
  • XML Output
  • Simple HTML Template
  • Layout HTML Template (Layout + Nested HTML)
  • Snippets of HTML can be included in multiple pages
  • Ability to call defined functions from HTML
