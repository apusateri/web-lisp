Layout HTML

This is the layout file used to render the HTML page.
Dynamic replacement is available within the layout, i.e.:
The result of (+ 4 6) is ~{ (+ 4 6) ~}
This is where we enter the "content" of the layout:
~{ body ~}
And then we can add more things in our layout afterwards ~{ (web-lisp:powered-by) ~}
